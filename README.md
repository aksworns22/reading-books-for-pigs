## 🎯 목적

## 💸 벌금 규칙

주당 정해진 분량을 읽지 못할 경우 4,000원 벌금을 내야합니다.

중도에 포기한다면 돼지고기 회식에 참여하지 않더라도 카드는 긁으러 오셔야합니다!

정해진 기간동안 완독에 실패한다면 20,000원의 벌금이 추가됩니다 💸
